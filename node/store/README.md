## Start
```npm install``` and ```npm start```

### Using http tool
ADD -> send POST with JSON:
```
    {
        "key" : "anyKey",
        "value" : "anyValue"
    }
```

SEARCH -> send GET with JSON:
```
    {
        "key" : "anyKey",
    }
```

PUT -> send PUT with JSON:
```
    {
        "key" : "anyKey",
        "value" : "anyValue"
    }
```

DELETE -> send DELETE with JSON:
```
    {
        "key" : "anyKey",
    }
```
